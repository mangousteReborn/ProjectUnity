Bonjour,


Etapes à suivre pour lancer la Démo.

[PRE-GAME]
- 1 / Lancer la release "Player" : A chaque lancement d'une realease, un nouveau Joueur rejoindra la partie. Le premier a se connecter sera le Server.

- 2 / Lancer la release "GameMaster" *1.

- 2.bis / Pour test le jeu sans le GameMaster, il suffit de connecter 2 Joueurs (cf. étape 1).

[IN-GAME]
- 3  /  Les joueurs sont en "Repos". C'est la partie ou ils choisissent leurs bonus. Ils sont en déplacement Libre.

- 4 / Lorsque les 'n' joueurs ont passé la porte, le jeu rentre en mode "Combat". Il n'est plus possible de changer les Bonus. On planifie les Actions.

- 5 / Lorsqu'un Joueur est prêt, il valide (bouton "Go :D"). Une fois que les 'n' Joueurs
ont validé, la simulation est lancé. Le jeu rentre en mode "Spectateur". Aucune action n'est possible.

- 6 / Lorsque toutes les Piles d'Actions sont terminée. Le jeu repasse en mode "Combat". Retour à l'étape 4.




*1 /!\ La partie GameMaster étant encore en construction, il est fortement déconseiller de lancer le comabat lorsqu'un GameMaster est dans la partie.


/!\ Ce qui ne marche pas :
- Le lancement des AI pour le gameMaster (elles sont codées mais ne s'exécute pas...).
- La condition d'attaque de l'Action Attaquer ("Traquer", "Sniper") des Joueurs. Les joueurs s'attaquent entre eux (Pour les tests, vu que les AI ne sont pas fonctionnels).
